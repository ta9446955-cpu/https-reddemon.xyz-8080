[1.0.1] - 2026-07-09
Fixed
- The Account settings category (Client ID, Client Secret, Sign-In Status) wasn't showing up at all in the Settings screen - only Display was visible. Caused by an invalid <hidden> element used to try to mask password input, which isn't valid Kodi schema and was breaking the whole category. Removed it; fields are now plain visible text, but Settings works correctly.

[1.0.0] - 2026-07-09
Initial release
- Sign in with your YouTube account using Google's device sign-in flow (no typing passwords into Kodi)
- Browse your own YouTube playlists
- Browse videos within each playlist, with pagination for large playlists
- Playback is delegated to the official YouTube addon (plugin.video.youtube) for reliable, up-to-date video streaming
- Sign out option available from the main menu
